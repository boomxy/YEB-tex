临近年末，当税务局提醒纳税人及时更新2020年个税专项附加扣除信息的同时，年终奖也临近了。

　　2019年是新个税法全面落地的第一年，综合收入以年为单位计税，还引入了子女教育、大病医疗、继续教育、房租、房贷利息、赡养老人等专项附加扣除，个税减免力度加大。为了稳定社会预期，全年一次性奖金（俗称年终奖）单独计税的政策，延续到2021年12月31日。

　　在减税力度加大的背景下，年终奖选择“单独计税”，还是并入当年综合收入“一并计税”，才能更好地获得减税红利，成为一道现实的选择题。

　　税务总局提示，对部分中低收入者而言，如将全年一次性奖金并入当年工资薪金所得，扣除基本减除费用、专项扣除、专项附加扣除等后，可能根本无需缴税或者缴纳很少税款。在此情况下，如果将全年一次性奖金采取单独计税方式，反而会产生应纳税款或者增加税负。同时，如单独适用全年一次性奖金政策，可能在税率换档时出现税负突然增加的“临界点”现象。

　　哪些人采用一并计税方式更佳？

　　比如，庞红作为工薪阶层，在某省会城市工作，月工资10000元。目前处于租房阶段，作为独生子女的庞红上有年满60岁的父母，2019年年度可获得年终奖3万元。

　　庞红的月度应纳税所得=10000-5000（基本减除费用，即起征点）-2500（专项扣除，假定社保和公积金合并缴费比例为25%）-1500（专项附加扣除，房租项）-2000（专项附加扣除，赡养老人项）=-1000元

　　这种情况下，庞红每月不用预交个税，还“积累”有扣除费用。对于庞红这类群体而言，将年终奖并入综合收入会更划算。

　　如果将年终奖并入年度收入计税，适用综合所得税率表（如下），

![img](http://n.sinaimg.cn/spider20191211/223/w700h323/20191211/0204-ikrsesr7249547.jpg)

　　庞红的全年应纳税所得=120000（年度工薪所得）+30000（年终奖）-60000（基本减除费用）-30000（社保、公积金等专项扣除）-42000（专项附加扣除，房租+赡养老人项）=18000元，全年个税纳税额度=18000*3%=540元。

　　如果将年终奖单独计税，适用全年一次性奖金过渡政策，即以全年一次性奖金收入除以12个月得到的数额，按照如下“按月换算后的综合所得税率表”，确定适用税率和速算扣除数，单独计算纳税。其中，应纳税额=全年一次性奖金收入×适用税率-速算扣除数。

![img](http://n.sinaimg.cn/spider20191211/254/w700h354/20191211/ca02-ikrsesr7249554.jpg)

　　庞红3万元年终奖（按月换算后对应“不超过3000元的”的级次）税率为3%，速算扣除数为0，年终奖应纳税额=30000*3%=900元。

　　这种情况下，年终奖并入全年收入只需缴纳个税540元，而单独计税需缴纳个税900元。对于庞红而言，年终奖单独计税的过渡政策，完全谈不上是优惠政策了。

　　哪些人适合单独计税方式？

　　唐亚作为工薪阶层，目前在一线城市工作，月工资2万元，目前背负有房贷，有兄弟姐们，上有年满60岁的父母。2019年度可获得年终奖2万元。

　　唐亚全年应纳税所得=240000-60000（基本减除费用）-60000（假定社保和住房公积金缴费比例为25%）-12000（房贷利息专项附加扣除）-12000（假定其分担了1000元/月赡养父母专项附加扣除）=96000元

　　按综合所得税率表，唐亚适用10%这档税率，年度应纳个税额度=96000*10%-2520=7080元。

　　如果将年终奖纳入年度收入一并纳税，对应全年应纳税所得额11.6万元，适用10%这档税率，唐亚年终奖应纳个税=20000*10%=2000元。

　　如果年终奖单独计税，适用3%这档税率，唐亚年终奖应纳个税=20000*3%=600元。

　　此外，年终奖不平滑的应纳税额曲线，在部分临界点时，会发生税款的“突变”，纳税人需要自行计算应纳税额，选择更优惠的计税方式。

　　这些年终奖的临界点包括3.6万元、14.4万元、30万元、42万元、66万元、96万元。

　　比如年终奖36000元，按照上述税率表，若单独计税，对应税率3%，需缴纳税款1080元。年终奖在此基础上增加1元，即年终奖36001元对应税率跃升为10%，需缴纳税款增加到3390.1元。年终奖增加1元，税款增加了2310.1元。

　　预留一定灵活空间

　　“年终奖是公司根据员工全年表现、经营业绩等发放给员工的一次性奖金，是否发放、什么时候发放、如何发放等，都是根据企业财务制度来确定。年终奖单独计算纳税的过渡政策，为企业和纳税人预留了一定灵活操作的空间。单独计税，还是合并计税，需要视具体情况而定。”智方圆税务师事务所主管合伙人王冬生对21世纪经济报道记者表示。

　　北京国家会计学院教授李旭红对21世纪经济报道记者表示，对于年终奖的认定，税法上称为“全年一次性奖金”，一年只能选择一笔收入，遵循真实性的原则。如果连基本扣除都没有做到应扣尽扣，选择与工薪收入合并计税，可能会更划算；如果平常工资金额比较高，年终奖单独计税，对纳税人而言可能会更有利。

　　为了确保新个税法顺利平稳实施、稳定社会预期，年终奖单独计税的政策延续到2021年底。年终奖提供了两种可供选择的计税方式，不过具体如何操作，选择哪种计税方式，还要看企业更具体的财务管理制度。

　　广州一家企业财务负责人对21世纪经济报道记者表示，他们公司目前不会考虑员工薪酬是高还是低，年终奖会采用单独计税的方式来进行。该负责人同时表示，他们公司员工平均薪酬在1万元以上，相较而言单独计税会更划算。
